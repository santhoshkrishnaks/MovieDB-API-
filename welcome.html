Welcome to MovieDB-API-

This contain one database with two models one for user list and other one is for movie list.

To Access User model

Website url followed by /user

To Access Movie model

Website url followed by /movie

Features

  1. You can get the user and movie by their user name and movie name
  2. You can post the user and movie in database
  3. You can update the user and movie in database by their names
  4. You can delete the user and movie in database by their names

To Get,Update,Delete The Data By Their Names

  1. Website url followed by /movie/name/moviename
  2. Website url followed by /user/name/username
